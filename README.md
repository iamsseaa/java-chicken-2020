# 메인 화면
- [ ] 원하는 기능 선택 출력
    - [ ] 주문등록
    - [ ] 결제하기
    - [ ] 프로그램 종료

## 주문등록 시 발생되는 화면

    ### 테이블 목록
    - [ ] 테이블 목록 출력
        - [ ] 어떤 테이블에서 결제 이전인 주문 등록이 있다면 ₩ 표시

    ### 테이블 선택
    - [ ] 유저가 테이블 선택

    ### 메뉴 선택
    - [ ] 전체 메뉴 출력 (치킨 6개, 음료 2개)
    - [ ] 유저가 등록할 메뉴 선택

    ### 메뉴 수량 입력
    - [ ] 메뉴의 수량 입력

## 결제하기 시 발생되는 화면

    ### 테이블 목록
    - [ ] 테이블 목록 출력
        - [ ] 어떤 테이블에서 결제 이전인 주문 등록이 있다면 ₩ 표시

    ### 테이블 선택
    - [ ] 유저가 테이블 선택

    ### 주문내역 출력
    - [ ] 해당 테이블의 주문내역 출력

    ### 결제 진행
    - [ ] /번 테이블의 결제를 진행합니다.
    - [ ] 결제 방법 선택
        - [ ] 1, 신용 카드
        - [ ] 2, 현금

    ### 최종 결제할 금액
    - [ ] 치킨 종류 메뉴가 10개 이상 시 10,000원 할인
    - [ ] 현금 결제 시 5% 할인 (중복 가능)
